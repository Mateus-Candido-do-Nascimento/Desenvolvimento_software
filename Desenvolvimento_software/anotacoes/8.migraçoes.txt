Passo 4: Aplicar migrações
Antes de continuar, vamos aplicar as migrações do banco de dados:

python manage.py migrate

O que são migrações?

Migrações: Scripts que criam/alteram estrutura do banco de dados

Por que aplicar?:

 Django precisa criar tabelas para funcionalidades básicas
Apps incluídos: admin, auth, contenttypes, sessions

Conceito: Sistema de Migrações do Django

O que acontece?

Django cria tabelas no banco de dados
Tabelas para usuários, sessões, admin, etc.
Banco SQLite será criado automaticamente


Por que é importante?

Sem migrações, o projeto não funciona completamente
É o primeiro passo para usar funcionalidades do Django
