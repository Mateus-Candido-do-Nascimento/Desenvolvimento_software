🔍 Conceito 1: O que é Django?
Django é um framework web Python que segue o padrão MVC (Model-View-Controller):
Model: Define a estrutura dos dados (banco de dados)
View: Lógica de negócio (o que acontece quando alguém acessa uma página)
Template: Como os dados são apresentados (HTML)
